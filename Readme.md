# 🧶 Loja NSO Têxtil - E-commerce de Redes e Mantas

Este é um projeto de interface para uma loja virtual focada no segmento têxtil, especializada em redes de descanso, mantas para sofá, toalhas e fios. O objetivo é criar uma experiência de compra visualmente agradável e funcional para o usuário.

---

### 🚀 Sobre o Projeto
Atualmente, o projeto está em sua **fase inicial**, focado na estruturação e estilização visual. É um excelente laboratório para aplicar conceitos de design responsivo e acessibilidade.

---

### 🛠️ Tecnologias Utilizadas
- **HTML5:** Estruturação semântica do conteúdo.
- **CSS3:** Estilização, layout (Flexbox/Grid) e responsividade.
- **Font Awesome:** Ícones para redes sociais e interface.

---

### 📋 Funcionalidades em Desenvolvimento
- [x] Cabeçalho com Menu de Navegação.
- [x] Banner principal de destaque.
- [ ] Vitrine de produtos (Redes, Mantas e Toalhas).
- [ ] Carrinho de compras (Visual).
- [ ] Rodapé com informações de contato e redes sociais.
- [ ] Página de detalhes do produto.

---

### 📈 Próximos Passos (Evolução Tecnológica)
Este projeto seguirá um cronograma de estudos para implementação de novas camadas de inteligência:
1. **Fase 1:** Implementação de interatividade com **JavaScript** puro (filtros de busca e menus dinâmicos).
2. **Fase 2:** Migração da interface para **ReactJS** para ganho de performance e componentização.
3. **Fase 3:** Desenvolvimento do Backend com **Node.js** para gestão de pedidos e estoque.
